## How to Run the Integrated Application

### Backend Setup
1. Navigate to the project root: `cd D:\HACKATHOUN2\phase2`
2. Install dependencies: `pip install -r requirements.txt`
3. Set up environment variables in `.env`
4. Initialize the database: `python init_db.py`
5. Run the backend: `uvicorn main:app --reload`

### Frontend Setup
1. Navigate to the frontend directory: `cd D:\HACKATHOUN2\phase2\frontend`
2. Install dependencies: `npm install`
3. Set up environment variables in `.env.local`
4. Run the frontend: `npm run dev`

The application is now fully integrated with secure authentication, proper user isolation, and a responsive user interface. Users can register, log in, and manage their tasks securely with all data properly isolated at the database level.
